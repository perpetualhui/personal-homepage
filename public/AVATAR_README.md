# 头像图片说明

请将你的头像图片放到这个文件夹（`public/`）中。

## 📝 要求

1. **文件名**: `avatar.jpg` 或 `avatar.png`
2. **尺寸**: 至少 400x400 像素（建议正方形）
3. **格式**: JPG 或 PNG
4. **大小**: 小于 2MB

## 📥 位置

将图片放到：`public/avatar.jpg`

## ⚠️ 注意

- 确保文件名是 `avatar.jpg` 或 `avatar.png`
- 不要放在子文件夹中
- 直接放在 `public/` 根目录

## 🎯 示例

正确：
```
public/
  └── avatar.jpg  ✅
```

错误：
```
public/
  └── images/
      └── avatar.jpg  ❌
```

## 📸 推荐的头像来源

- 专业照片
- 职业照
- 简洁的卡通头像
- 抽象设计

**完成后，你的头像就会显示在页面上了！**
